# cozy-sway-academy
A cozy (very spring like) rice for SwayWM, done for my new Fedora Linux Sway install

Welcome to another Linux Rice, this time my second Sway Rice, (if we dont count the private catppuccin rice i have lol) made for my new Fedora Sway install (i left gentoo) so enjoy and please star the repo if you like it!

Dependencies:
- SwayWM (SwayFX can even work, tho i havent tested and errors can occur)
- Waybar (the cool bar)
- Wofi (Wayland version of wofi that i prefer better than rofi tbh)
- Alacritty (Terminal, i love ini and toml configs.)
- Fastfetch (everyone has this one lol)
- Swaybg (?) (Wallpaper)
